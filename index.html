Dentro del universo de aplicaciones que existen para desarrollar paginas webs, es importante comenzar explicando lo que significa:

Diseño Web

Se refiere a la parte estética de una página web, es decir a los colores, botones, apariencia, todos los elementos visuales de una página o sitio.

Aplicaciones WEB

  • La web era al principio un conjunto de páginas fijas.
  • La evolución de la web ha llevado al desarrollo de aplicaciones web.
  • Una aplicación web es un programa informático en la red (en un servidor) cuya interacción con el usuario sea mediante un navegador web.
  • El programa recibe las entradas del usuario.
  • El programa genera dinámicamente páginas web que muestran información personalizada y en su caso permiten al usuario seguir interactuando.
El Diseño Web es un proceso o disciplina que se ocupa de la planificación, creación e implementación de páginas web, es decir, el diseño de interfaces y plataformas digitales.

Algunos de los elementos que hacen posible el Diseño Web son:
  • El diseño de la interfaz.
  • El diseño del material.
  • Los recursos gráficos.
  • La experiencia de usuario.
  • La Adaptabilidad del diseño en todo tipo de dispositivos.
El Diseño Web es lo que permite a millones de personas, proyectos y empresas a tener presencia en Internet y llegar a sus consumidores de la mejor forma.

Los elementos principales que incluyen el servicio de diseño web son:
  • Dominio (nombre de la página).
  • Hosting (almacenamiento en un servidor).
  • Diseño de la página web (estilo colores, logo).
  • Web responsiva (se adapta a la pantalla de cualquier dispositivo).
  • Accesibilidad (que sea práctica y que cargue rápido).
  • Auto gestionable (que el usuario pueda actualizar información).
  • Posicionamiento básico (para la búsqueda eficaz en Google).
